28/// Calculates the cube root of a number.
29///
30/// Parameters:
31///
32/// * `self` : The number for which to calculate the cube root.
33///
34/// Returns the cube root of `self`.
35///
36/// Special Cases:
37///
38/// * Return `NaN` if `self` is `NaN`.
39/// * Return `±0` if `self` is `±0`.
40/// * Return `Infinity` if `self` is `Infinity`.
41/// * Return `-Infinity` if `self` is `-Infinity`.
42///
43/// Example
44///
45/// ```moonbit
46/// test "cbrt" {
47///   inspect!(1.0.cbrt(), content="1")
48///   inspect!(3.0.cbrt(), content="1.4422495703074083")
49///   inspect!((-3.0).cbrt(), content="-1.4422495703074083")
50///   inspect!(10.0.cbrt(), content="2.154434690031884")
51///   inspect!(1000.0.cbrt(), content="10")
52///   inspect!(@double.not_a_number.cbrt(), content="NaN")
53///   inspect!(@double.infinity, content="Infinity")
54///   inspect!(@double.neg_infinity, content="-Infinity")
55/// }
56/// ```
