24/// Calculates the the square root of the sum of the squares of its arguments.
25///
26/// Parameters:
27///
28/// * `self` : The number to be used as the first argument.
29/// * `y` : The number to be used as the second argument.
30///
31/// Returns the hypotenuse of a right-angled triangle whose legs are `self` and `y`.
32///
33/// Example:
34///
35/// ```moonbit
36/// test "hypot" {
37///   inspect!(3.0.hypot(4.0), content="5")
38///   inspect!(5.0.hypot(12.0), content="13")
39///   inspect!(8.0.hypot(15.0), content="17")
40///   inspect!(7.0.hypot(24.0), content="25")
41///   inspect!(@double.not_a_number.hypot(1.0), content="NaN")
42///   inspect!(1.0.hypot(@double.not_a_number), content="NaN")
43///   inspect!(@double.infinity.hypot(1.0), content="Infinity")
44///   inspect!(1.0.hypot(@double.infinity), content="Infinity")
45///   inspect!(@double.neg_infinity.hypot(1.0), content="Infinity")
46///   inspect!(1.0.hypot(@double.neg_infinity), content="Infinity")
47/// }
48/// ```
