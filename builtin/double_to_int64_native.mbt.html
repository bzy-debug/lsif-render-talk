19/// Converts a double-precision floating-point number to a 64-bit integer.
20/// Handles special cases including NaN and numbers outside the valid Int range.
21///
22/// Parameters:
23///
24/// * `self` : The double-precision floating-point number to be converted.
25///
26/// Returns an 64-bit integer value according to the following rules:
27///
28/// * Returns 0 if the input is NaN
29/// * Returns `@int64.max_value` (9223372036854775807L) if the input is greater than or
30///   equal to `@int64.max_value`
31/// * Returns `@int64.min_value` (-9223372036854775808L) if the input is less than or equal
32///   to `@int64.min_value`
33/// * Otherwise returns the integer part of the input by truncating towards zero
34///
35/// Example:
36///
37/// ```moonbit
38/// test "Double::to_int64" {
39///   inspect!(42.0.to_int64(), content="42")
40///   inspect!((-42.5).to_int64(), content="-42")
41///   inspect!((0.0 / 0.0).to_int64(), content="0") // NaN
42///   inspect!((1.0 / 0.0).to_int64(), content="9223372036854775807") // Infinity
43///   inspect!((-1.0 / 0.0).to_int64(), content="-9223372036854775808") // -Infinity
44/// }
45/// ```
