16/// Copies a slice of elements from one fixed array to another.
17///
18/// This function copies `len` elements from `src` starting at `src_offset` to `dst` starting at `dst_offset`.
19/// The arrays may overlap, in which case the copy is performed in a way that preserves the data.
20///
21/// # Example
22/// ```
23/// let src = FixedArray::from_array([1, 2, 3, 4, 5])
24/// let dst = FixedArray::from_array([0, 0, 0, 0, 0])
25/// FixedArray::unsafe_blit(dst, 0, src, 0, 3)
26/// assert_eq!(dst, FixedArray::from_array([1, 2, 3, 0, 0]))
27/// ```
28///
29/// The behavior is undefined and platform-specific if:
30/// - `len < 0`
31/// - `src_offset < 0`
32/// - `dst_offset < 0`
33/// - `dst_offset + len > dst.length()`
34/// - `src_offset + len > src.length()`
35///
